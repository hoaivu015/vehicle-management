Hệ Thống Lưới Mềm & Khoảng Thở

Quy chuẩn khoảng cách dựa trên hệ số Fluid Soft Grid (bội số 8px), tối ưu hóa các không gian thở lớn (p-8 / 32px) để giảm tải căng thẳng nhận thức cho người dùng.

Hệ Thống 4 Lớp Chiều Sâu

Sử dụng chất liệu kính bán trong suốt (Specular Glassmorphism) kết hợp đường viền cực mảnh (Hairline borders) thay vì đổ bóng dày bẩn, tạo chiều sâu 3D sang trọng cho ứng dụng:

Layer 3: Tiêu điểm tối cao (Gemini Voice Live, Toasts, AI Blob) Solid/Glow
Layer 2: Phần tử tương tác nhỏ (Pills, Buttons, Inputs, Indicators) Opaque
Layer 1: Mặt thẻ / Khung chứa (Product Card, Dashboard Widgets) Blur-xl 70%
Layer 0: Mặt nền Canvas cơ bản (Ambient Gradient Flow) Base 100%
Ⅴ

Tối Ưu Phần Cứng (Hardware Acceleration)

Để đảm bảo các hiệu ứng mờ kính siêu nặng và chuyển động lò xo mượt mà 120Hz trên các thiết bị di động, mọi phần tử hoạt họa bắt buộc phải được đẩy lên Graphic Layer xử lý bởi GPU, triệt tiêu việc repaint của CPU: 

/* Ép trình duyệt tạo Graphics Layer riêng biệt */
.expressive-hardware-card {
  will-change: transform, backdrop-filter; 
  transform: translateZ(0);
}

/* Cô lập layout tránh vỡ layout-shift khi cuộn */
.expressive-grid-container {
  contain: layout style;
}
